Exceptions¶

exception get_vat_eu.VIESServiceError¶

The remote API is fauly for some reason.

exception get_vat_eu.ResponseIOError¶

Input and expected output differ.

exception get_vat_eu.ResponseVatNumberNotConforming¶

The VAT number returned by the API does not conform to specifications.

exception get_vat_eu.CannotGetTraderName¶

The API does not know what the address.

exception get_vat_eu.CannotGetTraderAddress¶

The API does not know what the address is.

exception get_vat_eu.VatNotValid¶

The input VAT is not valid.

exception get_vat_eu.AddressStringNotCorrespondingToExpectedFormat¶

The address string does not correspond to the expected format.

exception get_vat_eu.CountryCodeNotImplemented¶

The specified country has no implementation yet.
